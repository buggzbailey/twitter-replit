# django-twiiter
Check out [LIVE DEMO](https://django-twitter-t.herokuapp.com/) here!!
## Tech used
```
* Html
* Css
* Bootstrap
* Javascript / JQuery
* Django
* Cloudinary (To host uploaded images on CDN)
```
## User Story
```
* Users can post tweets.
* Users can edit tweets.
* Users can delete tweets.
* Users can send a like to a tweet.
```
